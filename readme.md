## MUSIC-PLAYER 🎶🎵

A simple yet engaging music player built with HTML, CSS, and JavaScript. 
It showcases the Milano-Cortina Winter Olympic theme song for Korea, 
allowing users to play, pause, and loop the track in an interactive interface.Designed to showcase both technical skill and cultural creativity, it transforms a simple player into a vibrant celebration of sport and music.

--------------------------------------

## Features 🚀

- Play / Pause / Stop Controls

- Loop Functionality → continuous replay of the anthem.

- Volume Control & Mute → adjustable sound levels.

- Seek Bar / Progress Indicator

--------------------------------

2. **CSS**
